### Учебный проект "Мессенджер" курса "Мидл фронтенд-разработчик"

[philomel.netlify.app](https://philomel.netlify.app/)

#### Устанока и запуск для разработки
```shell
npm i
npm run dev
```

#### Хостинг на сервере
```shell
npm run start
```

#### Продакшн сборка
```shell
npm run build
npm run prewiew
```
